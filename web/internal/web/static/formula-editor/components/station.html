Station

The DAG node. Its rarity frame is computed from the parsed TOML — never hand-assigned: gate → legendary, more than one needs → epic, exactly one → rare, none → common. Board positions come from longest-path ranking; this page lays them statically only to show states.

Rarity states

.station.framed.r-common — 0 needs

.station.framed.r-rare — 1 need

.station.framed.r-epic — 2+ needs

.station.framed.r-legendary — has gate

Interaction states

.selected — magenta ring

.error — danger ring, never magenta

.synthline — convoy formulas only

Belt edges are SVG paths (.belt-edge) tinted by the downstream station's rarity; the Test Fire spark (.spark) is the only moving object on the board and turns gold at gates, red at the first fault.
